# 公司账号资产管理系统 V1 — 设计原型交付报告

## 项目概述

基于 PRD 文档（account-asset-management-prd.md），为公司内部账号资产管理系统完成从需求发现到设计原型交付的全流程。

## 设计流程

| 阶段 | 参与成员 | 产出 |
|------|---------|------|
| Phase 1 需求发现 | 许明需 (discovery-analyst) | 五要素需求摘要（场景/受众/调性/品牌/规模） |
| Phase 2 设计系统选择 | 彩格调 (design-system-expert) | 推荐 3 套候选 → 用户选定 **Linear** → 生成 DESIGN.md |
| Phase 3 原型生成 | 筑原型 (prototype-builder) | 完整 HTML/CSS/JS 原型（5 页面 + 面板 + 弹窗） |
| Phase 4 质量审查 | 严过审 (critique-reviewer) | 5 维评审 + Anti-Slop 检测 → REVISE → 修复 → PASS |
| Phase 5 导出交付 | 交付达 (export-specialist) | 最终 HTML 交付物（103KB，独立可运行） |

## 关键设计决策

### 设计系统：Linear 风格
- **选择理由**：表格+右侧面板交互模式高度同构、蓝灰专业色系 + 红色克制使用、权限感知 UI 与 5 种角色体系天然匹配
- **视觉哲学**：极简功能主义 × 数据精准 — 用克制的蓝灰层次承载密集信息

### 三栏布局
- 左侧导航 (228px) → 中间主工作区 → 右侧详情面板 (520px)
- Core：搜索框 → KPI 指标条 → 视图 Tabs → 筛选栏 → 批量操作栏 → 数据表格 → 分页

## 原型功能清单

- ✅ **4 个页面**：账号资产主页 / 操作日志 / 风险提醒 / 系统设置
- ✅ **三栏布局**：左侧导航 + 中间表格 + 右侧详情面板
- ✅ **14 列数据表格**：固定表头 + 首列固定 + 横向滚动
- ✅ **全局搜索**：300ms 防抖，覆盖 10+ 字段
- ✅ **6 种视图切换**：全部/手机号/平台账号/关系缺失/高风险/最近修改
- ✅ **KPI 指标条**：5 个关键指标
- ✅ **6 维筛选栏**：类型/状态/风险/部门/负责人/使用人
- ✅ **批量操作栏**：选择后出现，4 种批量操作
- ✅ **右侧详情面板**：9 个信息区块，滑入动画
- ✅ **敏感信息脱敏**：默认 ••••••，查看/复制均写日志，30s 自动恢复
- ✅ **新增/编辑弹窗**：5 必填 + 15 非必填字段
- ✅ **分页组件**：默认每页 50 条
- ✅ **3 种角色切换**：管理员 / 普通员工 / 审计员（权限感知 UI）
- ✅ **响应式布局**：Tablet 断点 1024px 以下侧栏折叠
- ✅ **空状态设计**：搜索无结果、表格无数据
- ✅ **骨架屏**：初始加载 + 面板打开时显示 shimmer 动画
- ✅ **Toast 反馈**：操作结果提示
