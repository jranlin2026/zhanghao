# 公司账号资产管理系统 V1（修正版）— 设计原型交付报告

## 概述

基于用户反馈对首次原型进行了全面修正：
1. **业务模型重构**：从扁平账号表改为「手机设备 → 手机号 → 互联网账号」三层递进结构
2. **UI 布局修正**：固定三栏永不折叠，去掉了响应式折叠侧栏

## 设计流程

| 阶段 | 参与成员 | 产出 |
|------|---------|------|
| Phase 1 需求发现 | 许明需 (discovery-analyst) | requirements-revised.md 三层架构需求 |
| Phase 2 设计系统 | 沿用已选 **Linear** 风格 | 设计规范复用 DESIGN.md |
| Phase 3 原型生成 | 筑原型 (prototype-builder) | prototype-v2.html 三层原型（101KB） |
| Phase 4 质量审查 | 严过审 (critique-reviewer) | 5 维评审 3.8/5 **PASS** → 修复 7 项问题 |
| Phase 5 导出交付 | 交付达 (export-specialist) | 零改动，直接可交付 |

## 三层数据模型

```
手机设备（Device）
  ├── 单卡 → SIM1 卡槽 → 手机号
  │                       ├── 微信账号
  │                       ├── 抖音账号
  │                       └── ...
  └── 双卡 → SIM1 卡槽 → 手机号1
              │            ├── 微信账号
              │            └── ...
              └── SIM2 卡槽 → 手机号2
                             ├── 小红书账号
                             └── ...
```

## 原型功能清单

- ✅ **固定三栏布局**：左侧 220px 导航 | 中间自适应主区 | 右侧 520px 详情面板
- ✅ **三种视图 Tab**：设备列表（15 列）/ 手机号视图（9 列）/ 互联网账号视图（11 列）
- ✅ **15 行设备示例数据**：含单卡/双卡设备、对应手机号和互联网账号
- ✅ **智能右侧面板**：根据选中的实体类型（设备/手机号/账号）动态展示不同内容
- ✅ **面包屑导航**：设备名 > 手机号 > 账号名，支持点击向上跳转
- ✅ **全局搜索**：覆盖设备名/IMEI/手机号/账号名称等
- ✅ **KPI 指标条**：设备总数 / 使用中 / 闲置 / 高风险 / 无负责人
- ✅ **6 维筛选栏**：SIM 类型、平台类型、所属部门、负责人、状态、风险等级
- ✅ **批量操作栏**：选中后出现，收回权限/移交负责人/标记高风险/导出所选
- ✅ **新增设备弹窗**：7 必填 + 非必填，完整表单校验
- ✅ **敏感信息脱敏**：默认 ••••••，查看/复制写日志，30s 自动恢复
- ✅ **3 种角色切换**：管理员/普通员工/审计员
- ✅ **面板遮罩层**：半透明遮罩，点击关闭
- ✅ **面板动画**：visibility + transform 滑入滑出
- ✅ **风险左侧色条**：透明背景 + 4px 色条（红/黄/灰）
- ✅ **骨架屏 + 空状态 + Toast 通知**
- ✅ **操作日志页 / 风险提醒页 / 系统设置页**
